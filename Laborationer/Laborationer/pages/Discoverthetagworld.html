Tables

The< table > element is use to create tables. The < th > element shows the header cell in the table and it describes what the data stands for. It is the name of the colum and the row. The < tr > element shows the row in the table we are writing on followed by some < td > elements that is the table data. looks like this:

The < caption > element represent the title of the table. It has to be inside the table tag.

The < col > tag is useful for applying styles to entire columns, instead of repeating the styles for each cell, for each row

The < colgroup > tag specifies a group of one or more columns in a table for formatting. The < colgroup > tag is useful for applying styles to entire columns, instead of repeating the styles for each cell, for each row. The < colgroup > tag must be a child of a < table > element, after any < caption > elements. To define different properties to a column within a < colgroup >, use the < col> tag within the < colgroup > tag.
